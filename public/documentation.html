API documentation

The API lets you search for movies. You can search by movie name, "genre" or "director" name.

Allow users to update their information /users/:username PUT JSON { "username": "sakiproudfoot", "password": "supersecretstuffhere", "email": "sakiproudfoot@gmail.com" } JSON { "username": "sakiproudfoot", "id": "qwer43532qwer4362" "password": "supersecretstuffhere", "email": "sakiproudfoot@gmail.com", "favoriteMovies": "[]" }
Allow users to add a movie to their favorites list /users/:username/movies/:title POST None /users/sakiproudfoot/movies/Day%20of%20the%20Dead JSON { "favoriteMovies": ["Day of the Dead"] }
Allow users to remove a movie from their favorites list /users/:username/movies/:title DELETE None /users/sakiproudfoot/movies/Day%20of%20the%20Dead JSON { "favoriteMovies": [] }
Allow existing users to deregister /users/:username DELETE None /users/sakiproudfoot text "sakiproudfoot has been deregistered"
